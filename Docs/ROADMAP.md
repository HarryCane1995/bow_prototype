# Roadmap

Ближайшие возможные задачи:

- Исправление/полировка crosshair.
- Звуки натяжения, выстрела и попадания.
- Втыкание стрел в мишени.
- Баллистика стрелы.
- Улучшение target feedback.
- Блокинг небольшой арены.
- Настройка movement feel.
- Улучшение bow viewmodel.
- Индикатор полного натяжения.

Документ намеренно лёгкий: это список ближайших направлений, а не строгий production-план.
