# Asset Pipeline

Текущий пайплайн ассетов:

`Blender -> glTF Binary .glb -> Godot`

Модель лука используется как FPS viewmodel. Визуальная модель нужна для ощущения натяжения, положения руки/лука и обратной связи при стрельбе.

## Важные имена

- `Bow_ViewModel`
- `Bow_Body`
- `BowRig`
- `Arrow_Visual`
- `String_Upper`
- `String_Lower`
- `Draw animation`

## Draw-анимация

`Draw animation` должна управляться значением charge amount от `0` до `1`:

- `0` - лук не натянут.
- `1` - лук полностью натянут.

Это значение должно использоваться для визуальной синхронизации лука с игровой логикой натяжения.

## Визуальная и projectile-стрелы

Визуальная стрела в луке не равна projectile-стреле.

- `Arrow_Visual` - часть bow viewmodel, нужна для отображения стрелы до выстрела.
- Projectile-стрела создаётся кодом отдельно и живёт как самостоятельный игровой объект.
